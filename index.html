Callejero de España

Buscador de códigos postales

Escribe o dicta una calle para conocer su código postal y verla en el mapa. Si escribes un código de 5 dígitos, te diré a qué localidad pertenece.

Datos oficiales de CartoCiudad (IGN). Si una calle existe en varias localidades, te mostraré la lista para que elijas.

⚠️ Atención: esta calle tiene más de un código postal según el número de portal. Indica el portal para obtener el código exacto.
